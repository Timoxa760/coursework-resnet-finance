## Описание

Практическая часть курсовой работы, посвящённой применению остаточных нейронных сетей (ResNet) для прогнозирования направления движения цен российских акций (SBER, GAZP, YNDX).

## Структура проекта

```
├── src/                    # Исходный код
│   ├── config.py           # Конфигурация
│   ├── data.py             # Загрузка данных
│   ├── features.py         # Feature engineering
│   ├── models.py           # ResNet, LSTM
│   ├── train.py            # Обучение
│   ├── evaluate.py         # Оценка и визуализация
│   └── utils.py            # Утилиты
├── notebooks/              # Jupyter-ноутбуки
│   └── 01_experiments.ipynb
├── data/                   # Данные (не включены в репозиторий)
├── results/                # Графики и метрики
├── requirements.txt        # Зависимости
└── README.md               # Этот файл
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

Откройте `notebooks/01_experiments.ipynb` в Jupyter и выполните ячейки по порядку.

## Модели

- **ResNetTimeSeries** — остаточная сеть с 1D-свёртками и skip-connections
- **LSTMClassifier** — baseline на основе LSTM
- **RandomForest** — классический baseline (sklearn)
